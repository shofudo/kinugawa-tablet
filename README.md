# きぬ川不動瀧 客室タブレット用インフォメーションサイト

## 📁 ファイル構成

```
kinugawa-tablet/
├── index.html          ← メインのHTMLファイル（基本的に編集不要）
├── styles.css          ← デザイン用CSSファイル（基本的に編集不要）
├── script.js           ← 動作用JavaScriptファイル（基本的に編集不要）
├── config.json         ← 🔄 このファイルを編集して情報を更新します
├── README.md           ← このファイル（使い方説明）
└── images/             ← 画像を入れるフォルダ
    ├── logo.png        ← 旅館のロゴ画像（後で差し替え）
    └── seasonal/       ← 季節のポスター画像を入れるフォルダ
        ├── spring-strawberry.jpg
        └── green-tour.jpg
```

---

## 🔄 情報を更新する方法

### ステップ 1: `config.json` を開く

VS Code で `config.json` ファイルを開きます。

### ステップ 2: 変更したい箇所を編集

例えば、Wi-Fi のパスワードを変更する場合:

**変更前:**

```json
"wifi": {
  "networkName": "kinugawa-guest",
  "password": "fudotaki2025"
},
```

**変更後:**

```json
"wifi": {
  "networkName": "kinugawa-guest",
  "password": "新しいパスワード"
},
```

### ステップ 3: 保存する

`Ctrl + S` (Windows) または `Cmd + S` (Mac) で保存します。

### ステップ 4: サーバーにアップロード

GitHub Pages または Firebase にアップロードすると、全タブレットに反映されます。

## 📝 各項目の更新方法

### 1. Wi-Fi 情報の更新

```json
"wifi": {
  "networkName": "ネットワーク名",
  "password": "パスワード"
}
```

### 2. 館内設備の更新

**製氷機の場所を変更:**

```json
"iceMachine": "3階 エレベーター横"
```

**コーヒーメーカーの使い方を変更:**

```json
"coffeeMaker": {
  "location": "",
  "steps": [
    "手順1",
    "手順2",
    "手順3"
  ]
}
```

### 3. レンタサイクル情報の更新

```json
"bicycle": {
  "location": "1階 フロント横",
  "hours": "9:00〜17:00",
  "price": "2時間 500円 / 1日 1,000円"
}
```

### 4. お飲み物情報の更新

```json
"drinks": {
  "description": "お飲み物の説明文",
  "lastOrder": "22:00"
}
```

### 5. お風呂情報の更新

```json
"bath": {
  "name": "大浴場「渓流の湯」",
  "hours": {
    "evening": "15:00〜23:00",
    "morning": "6:00〜9:30"
  },
  "notes": [
    "注意事項1",
    "注意事項2"
  ]
}
```

### 6. 季節のご案内の更新 🌸

**新しいイベントを追加する場合:**

```json
"seasonal": [
  {
    "title": "イベントタイトル",
    "period": "2025年3月1日〜5月31日",
    "description": "イベントの説明文",
    "image": "images/seasonal/ファイル名.jpg"
  }
]
```

**画像の追加方法:**

1. `images/seasonal/` フォルダに画像ファイルを入れる
2. ファイル名を `config.json` の `"image"` に書く

**イベントを削除する場合:**
該当する `{ }` ブロック全体を削除します。

### 7. 周辺観光情報の更新

```json
"sightseeing": [
  {
    "name": "施設名",
    "distance": "当館より車で約5分",
    "comment": "一言コメント"
  }
]
```

## 🖼️ 画像の追加・差し替え方法

### 季節のポスター画像

1. `images/seasonal/` フォルダに画像を入れる
2. ファイル名は英数字で（例: `spring-2025.jpg`）
3. `config.json` の該当箇所に画像パスを記入

**推奨画像サイズ:**

- 縦長の A4 サイズ（例: 600px × 800px）
- ファイル形式: JPG または PNG
- ファイルサイズ: 1MB 以下を推奨

### ロゴ画像の差し替え

1. `images/logo.png` を旅館のロゴ画像に差し替える
2. ファイル名は `logo.png` のまま
3. 推奨サイズ: 横 500px 程度、背景透過 PNG

## 🚀 サーバーへのアップロード方法

### GitHub Pages の場合

1. VS Code で変更を保存
2. ターミナルで以下を実行:

```bash
git add .
git commit -m "情報を更新しました"
git push
```

3. 数分後に自動で反映されます

### Firebase の場合

1. VS Code で変更を保存
2. ターミナルで以下を実行:

```bash
firebase deploy
```

3. 数分後に反映されます

## 💡 よくある質問

### Q1. JSON ファイルの編集で注意することは?

**A1. 以下の点に注意してください:**

- ダブルクォーテーション `"` を消さない
- カンマ `,` を忘れない（最後の項目の後は不要）
- 全角スペースは使わない（半角スペースのみ）
- 日本語は OK です

### Q2. 画像が表示されません

**A2. 以下を確認してください:**

- 画像ファイルが正しいフォルダに入っているか
- ファイル名が `config.json` と一致しているか
- ファイル名に日本語や記号が含まれていないか

### Q3. 編集後、タブレットに反映されません

**A3. 以下を試してください:**

1. サーバーにアップロードしたか確認
2. タブレットのブラウザを更新（リロード）
3. キャッシュをクリア

### Q4. JSON ファイルを壊してしまいました

**A4. 以下の方法で確認できます:**

1. https://jsonlint.com/ にアクセス
2. `config.json` の内容をコピー&ペースト
3. エラー箇所を教えてくれます
