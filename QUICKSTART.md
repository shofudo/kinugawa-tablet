# 🚀 クイックスタートガイド

きぬ川不動瀧 客室タブレット用サイト - 最速で使い始める方法

---

## ⚡ 3ステップで始める

### ステップ1: ファイルをダウンロード
このフォルダをダウンロードして、適当な場所に保存します。

### ステップ2: VS Codeで開く
フォルダを VS Code で開きます。

### ステップ3: ブラウザで確認
`index.html` を右クリック → 「Open with Live Server」
（Live Server拡張機能がない場合は、ダブルクリックでもOK）

これだけで、ローカルで動作確認ができます!

---

## 📝 情報を更新したい時

### 1. `config.json` を開く

### 2. 変更したい箇所を編集

例: Wi-Fiのパスワードを変更
```json
"wifi": {
  "password": "新しいパスワード"
}
```

### 3. 保存（Ctrl + S または Cmd + S）

### 4. ブラウザを更新（F5）

これだけです!

---

## 🖼️ 画像を変更したい時

### 季節のイベント画像

1. `images/seasonal/` フォルダに画像を入れる
2. ファイル名を英数字にする（例: `spring2025.jpg`）
3. `config.json` に画像名を書く

```json
"seasonal": [
  {
    "title": "春のいちごフェア",
    "image": "images/seasonal/spring2025.jpg"
  }
]
```

### ロゴ画像

`images/logo.svg` を、旅館のロゴ画像に差し替えるだけです。
（ファイル名は `logo.png` または `logo.svg` にしてください）

詳しくは `DEPLOY.md` をご覧ください。

1. GitHubにアップロード

## 🎨 デザインを変更したい時

### 色を変更

`styles.css` を開いて、以下の色コードを変更:

```css
background-color: #F7F5F2;  /* 背景色 */
color: #4A7C7E;             /* メインカラー（緑） */
color: #C9A961;             /* アクセントカラー（ゴールド） */
```

## 📁 重要なファイル

| ファイル | 説明 | 編集 |
|---------|------|-----|
| `config.json` | **情報を更新するファイル** | ✅ よく編集する |
| `index.html` | ページの構造 | ❌ 基本触らない |
| `styles.css` | デザイン・色 | △ 色を変えたい時だけ |
| `script.js` | 動作プログラム | ❌ 基本触らない |
| `images/` | 画像フォルダ | ✅ 画像を追加する |

## 💡 覚えておくと便利

### JSONファイルの注意点

❌ ダメな例:
```json
"wifi": {
  "password": "test"  ← 最後にカンマがある（ダメ）
}
```

✅ 良い例:
```json
"wifi": {
  "password": "test"
}
```

### よくある間違い

1. **カンマを忘れる** → エラーになります
2. **クォーテーション `"` を消す** → エラーになります
3. **日本語ファイル名** → 画像が表示されないことがあります

### エラーチェック

`config.json` が正しいか確認:
https://jsonlint.com/ にコピペして「Validate JSON」
