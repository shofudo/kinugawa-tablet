## 📝 情報を更新する方法（超簡単）

### config.json を編集するだけ!

1. VS Codeで `config.json` を開く
2. 変更したい箇所を編集（例: Wi-Fiのパスワード）
3. 保存（Ctrl + S）
4. サーバーにアップロード

たったこれだけです!

## 🎨 カスタマイズ例

### Wi-Fiパスワードを変更
```json
"wifi": {
  "password": "新しいパスワード"
}
```

### 季節イベントを追加
```json
"seasonal": [
  {
    "title": "夏祭りイベント",
    "period": "2025年8月1日〜8月31日",
    "description": "浴衣でご来館のお客様には特別サービス!",
    "image": "images/seasonal/summer2025.jpg"
  }
]
```

### お風呂の時間を変更
```json
"bath": {
  "hours": {
    "evening": "15:00〜24:00",
    "morning": "5:30〜9:30"
  }
}
```
